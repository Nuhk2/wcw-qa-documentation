## ❓ TROUBLESHOOTING

### GitHub Pages Not Building?
1. Check "Settings" → "Pages" tab
2. Verify branch is set to `main`
3. Check "Actions" tab for failed builds
4. Wait 2-3 minutes (first build takes longer)

### Documents Not Rendering?
1. Verify `_config.yml` is in repo root
2. Ensure markdown files are in main branch
3. Try clearing browser cache (Cmd+Shift+R)

### Links Not Working?
1. Check file names exactly (case-sensitive on Linux)
2. Ensure `.md` extension is included in links
3. Use relative links: `../FILENAME.md`

### Can't See Updates?
1. Hard refresh browser: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
2. Wait 1-2 minutes for GitHub Pages to rebuild
3. Check "Actions" tab for build status
